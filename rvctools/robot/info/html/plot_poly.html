plot_poly

Plot a polygon

plotpoly(p, options) plot a polygon defined by columns of p which can be 2xN or 3xN.

OPTIONS

'fill' the color of the circle's interior, Matlab color spec
'alpha' transparency of the filled circle: 0=transparent, 1=solid.
